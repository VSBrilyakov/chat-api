# REST API чатов и сообщений
Небольшой веб-сервер, реализующий REST API для работы с чатами и сообщениями.

# В данном проекте реализовано следующее:
- Работа с net/http.
- Работа со структурными тегами.
- Подход Чистой Архитектуры в построении структуры приложения. Техника внедрения зависимости.
- Работа с БД Postgres. Использование GORM. Запуск из Docker.
- Генерация файлов миграций(goose).
- Написание SQL запросов.
- Использование testify и mock для написания unit тестирования.

### Предварительные требования
- Go v1.25.1+
- Docker
- Git

### Установка
```bash
git clone https://github.com/VSBrilyakov/chat-api
cd chat-api
```

## Для запуска приложения:
```bash
make run
```

### Если приложение запускается впервые, необходимо применить миграции к базе данных:
```bash
make migrate
```

## Для запуска тестов:
```bash
make test
```
